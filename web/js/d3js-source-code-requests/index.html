d3.js源码分析之Requests

2016-11-12

文件的加载对于很多应用都格外重要，对d3来说也是如此。对于绘制结构简单、数据量不大的图形，尚且可以将数据和js代码存放在一起，但是对于数据结构复杂、数据量庞大的情况，数据集应该被单独存放于独立的文件中，因此这里就涉及到了文件的读取问题。该模块用于对原生的XMLHttpRequest进行封装，可用于加载文件。
